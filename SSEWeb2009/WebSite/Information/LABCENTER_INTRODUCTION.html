同济大学软件学院简介

 

 

同济大学软件学院成立于2001年9月，是国家教育部与原国家计委联合批准成立的国家示范性软件学院。中国科学院院士、我国首台银河巨型计算机主机系统总设计师周兴铭教授任院长。

 

学院位于同济大学嘉定校区，院舍面积约8400平方米，现有各类在册学生2900余人。设有企业计算与软件技术、服务工程(大型主机技术)、软件工程与项目管理、图像处理与图形学、新媒体艺术与科学等多个专业方向。专业课程中20％ 课程与国外大学和知名企业合作，20%课程由资深企业技术专家授课，60% 课程可双语授课，使学生不仅有扎实的理论基础，还能掌握国际最新前沿技术和扎实的英文交流能力。建院8年来，软件学院学生先后获得国际竞赛大奖和全国竞赛大奖40余项。

 

学院拥有一支以专职教师为核心，专职与兼职相结合，高校与企业相结合、国内与国外相结合的结构多元化的特色师资队伍。学院先后聘请14位国外及港台地区的资深企业专家到学院授课，11位国内外资深专家受聘为同济大学兼职教授，4位外籍专家长期在学院工作，现有专业教师中有企业工作经历的13人，4人担任过企业技术主管、CTO、软件质量总监等高级职位并具有8年以上企业工作经验，40%的专职教师能胜任纯英文授课。

 

学院拥有一流教学和实验环境，现有包括IBM大型机和中小型商用服务器在内的各类主机10台、高配置PC机800多台、Apple计算机30台、其它实验装置180台（套），实验室免费向学生开放、计算机全部接入校园网、无线网络覆盖全院。

 

学院设有2个国际合作机构：International Software Testing Qualification Board (China)(国际软件测评委员会中国分会)、Chinese DB2 User Group（大中华区大型主机数据库用户协会(筹)；3个地区科教中心： IBM主机系统教育中心（上海）、IBM System i系统教学中心(上海)、上海超级计算中心同济分中心；7个联合技术中心：同济大学IBM技术中心、上海航天-同济大学航天嵌入式计算机技术实验室、微软-同济大学Windows移动与嵌入式中心、同济大学Apple开发技术实验室、同济大学AMD技术中心、同济大学Intel多核技术创新实验室、ICOP-同济大学Windows移动与嵌入式中心；7个学生科技创新俱乐部：同济大学Apple俱乐部、IBM俱乐部、Windows嵌入式俱乐部、Autodesk俱乐部、同济大学Google Camp、同济大学AMD俱乐部、同济大学Oracle俱乐部等。

 

学院致力于国际交流和合作，积极探索适合中国学生特点的培养具有国际合作与竞争能力的高层次软件人才培养模式。以互免学费、双向对等为特色的大规模国际交换学生项目为突破口，通过国际交换、国外实习和交流等多种灵活形式，近三年共派出近200名学生赴欧美等发达国家交流，并有近50名欧美高校的学生来我院交流学习，是国内少数的能向发达国家学生开放纯英文授课专业课的IT院校。

 

学院以市场需求为导向，主动加强与国内外企业的合作与中外合作，建院至今，已先后与100多家国内外IT企业建立合作关系，并与其中的近20家企业建立了实习基地。

 

2006年，软件学院顺利通过了教育部组织的示范性软件学院的终期验收评估，评估结果为“综合排名第五”。同济大学软件学院按照培养高层次、实用型、复合型、国际化人才为目标，秉承同济大学“同舟共济、求真务实”的传统，以坚持质量第一、素质与技术并重、理论与实践结合、可持续发展为建院宗旨，争取将学院建设成为有一定国际知名度的软件学院。
